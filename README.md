# 锱铢必较 (FairShare)

> 精准 AA 账单分摊工具

精准计算每个人应付的金额，支持按比例分配餐费。

---

## 功能特点

| 功能 | 说明 |
|------|------|
| 📷 **账单上传** | 上传用餐小票照片，为后续 AI 自动识别预留接口 |
| 🍔 **信息录入** | 录入每道菜的名称和价格，支持随时修改 |
| 🎚️ **比例分配** | 每人用滑块选择各道菜的食用百分比，超过 100% 会提示 |
| 🤷 **自动平摊** | 没人认领的部分自动均摊给所有人 |
| 📊 **结算明细** | 显示每人应付金额明细：认领部分 + 平摊部分 |

---

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/takerG/fair-share.git
cd fair-share

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开浏览器访问 `http://localhost:5173`。

## 使用流程

```
 ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
 │  1. 上传账单  │ ──▶ │  2. 添加信息  │ ──▶ │  3. 分配比例  │ ──▶ │  4. 查看结果  │

```

## 技术栈

- **Vanilla CSS** — Glassmorphism 主题风格
- **纯前端** — 数据保存在本地浏览器
